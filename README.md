A lightweight system monitoring tool built with Python for Linux systems. This application tracks key system metrics such as CPU usage, memory consumption, and network upload/download speeds in real time.

🚀 Features
  📊 Real-time CPU usage monitoring
  🧠 Memory (RAM) usage tracking
  🌐 Network activity (Upload & Download speed)
⚡ Lightweight and fast
🐧 Designed specifically for Linux systems
🛠️ Tech Stack
  Language: Python
  Platform: Linux
Libraries Used:
  psutil – for system and process monitoring
  time – for intervals and refresh rate
  OS- to Interact with the Operateing system
